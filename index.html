JavaScript

An object-oriented computer programming language commonly used to create interactive effects within web browsers.

Operators

An operator performs some operation on single or multiple operands (data value) and produces a result.

Variables

A value that can change, depending on conditions or on information passed to the program.

Variable Naming Convention

camelCase - First letter of the first word in the function name is lowercase, while all other words of the function name begin with a captial letter. Example: thisIsMyFunctionName()

Functions

A block of code designed to performa a particular task.

Methods

Actions that can be performed on objects.

Arguments

The real values passed to (and received by) the function when it is called.

Parameters

The names listed in the function definition. Serve as placeholders for arguments that will be passed into the function when called.
